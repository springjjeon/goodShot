# Good Shot - 골프공 궤적 추적기

이 프로젝트는 골프 스윙 비디오를 분석하여 골프공의 궤적을 자동으로 추적하고 시각화하는 Windows 데스크톱 애플리케이션입니다.

## 주요 기능
- 비디오 파일(.mp4, .avi 등)을 불러와 재생합니다.
- 영상 프레임에서 골프공의 위치를 수동으로 마크합니다.
- 마크된 공의 이동 경로를 계산하여 궤적을 그립니다.
- 궤적이 추가된 결과 영상을 보여줍니다.

## 기술 스택
- **언어**: Python
- **GUI**: PyQt5
- **영상처리**: OpenCV
- **데이터 처리**: NumPy, SciPy

## 현재까지 진행 상황
1. **프로젝트 환경 설정**: `requirements.txt` 파일 및 기본 폴더 구조 생성 완료.
2. **GUI 골격 구현**: 비디오 표시 영역과 기본 버튼을 포함한 메인 창 구현 완료.
3. **비디오 로드 기능**: '비디오 열기' 버튼으로 동영상 파일을 선택하고 첫 프레임을 화면에 표시하는 기능 구현 완료.
4. **자동 궤적 저장/로드**: 동영상 파일명별로 자동 저장하고 불러오는 기능 추가 완료.
5. **외부 내보내기/불러오기**: 궤적을 외부로 내보내고 불러오는 기능 추가 완료.

## 실행 방법

1. **필요 라이브러리 설치:**
    ```shell
    py -m pip install -r requirements.txt
    ```

2. **애플리케이션 실행:**
    ```shell
    py main.py
    ```

## 궤적 저장 시스템

### 자동 저장/로드
- 비디오를 로드하면 같은 폴더에서 자동으로 `{비디오파일명}_trajectory.json` 파일을 찾아 로드합니다.
- 궤적을 추가/수정/삭제할 때 자동으로 저장됩니다.
- 예: `shot1.mp4` → `shot1_trajectory.json` 자동 생성

### 외부 내보내기/불러오기
- **궤적 내보내기**: 현재 궤적을 원하는 경로의 JSON 파일로 저장
- **궤적 가져오기**: 외부 JSON 파일에서 궤적을 불러오기

## 사용 방법

### 기본 조작
- `[Space]` - 재생/일시정지
- `[◀]` - 이전 프레임
- `[▶]` - 다음 프레임
- `[분석 모드 켜기]` - 공 위치 마킹 모드 활성화
- `[마우스 클릭]` - 분석 모드에서 공의 위치 선택
- `[ESC]` - 현재 프레임의 좌표 삭제

### 궤적 관리
- **분석 모드 켜기**: 비디오의 각 프레임마다 골프공의 위치를 마우스로 클릭하여 표시
- **궤적 보기**: "궤적 전체 보기" 체크박스로 모든 궤적 표시/숨김
- **자동 저장**: 궤적은 수정될 때마다 자동으로 저장됨
- **내보내기/가져오기**: 버튼을 통해 궤적을 다른 위치로 이동하거나 공유 가능

## 개발 Action Plan

### Step 1: 비디오 연속 재생 기능 구현 (Video Playback) ✓
- `QTimer`를 사용하여 일정한 간격으로 다음 프레임을 읽어와 화면을 업데이트.
- '분석 시작' 버튼을 누르면 영상이 연속 재생되도록 뼈대 마련.

### Step 2: 골프공 객체 탐지 (Golf Ball Detection)
- 프레임 흑백 변환 및 블러 처리로 노이즈 감소.
- 배경 차분 또는 색상 필터링 기법으로 움직이는 물체 추출.
- 윤곽선(Contours) 검출 후 원형 객체를 골프공으로 판별.
- 탐지된 공 위치에 마커 표시.

### Step 3: 궤적 추적 및 시각화 (Trajectory Tracking) ✓
- 매 프레임 탐지된 골프공 중심 좌표를 리스트/데큐에 저장.
- 저장된 좌표들을 선으로 연결하여 스윙 궤적 시각화.
- 궤적 색상, 두께, 꼬리 길이 설정.

### Step 4: 성능 최적화 및 UI/UX 개선 (Advanced Features)
- 궤적 자동 저장/로드 시스템 구현 ✓
- 외부 내보내기/불러오기 기능 ✓
- **QThread**를 사용하여 영상 처리 로직을 백그라운드 스레드로 분리.
- 진행률 바(Progress Bar) 또는 상태 메시지 추가.
- 궤적이 그려진 최종 결과 영상을 파일로 저장하는 기능(cv2.VideoWriter) 추가.
